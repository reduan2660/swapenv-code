# swapenv for VS Code

Manage [swapenv](https://swapenv.sh) environments directly from the status bar.

## Requirements

- [swapenv CLI](https://github.com/reduan2660/swapenv) installed and available in PATH

## Usage

Click the status bar item or press `Cmd+Shift+S` (`Ctrl+Shift+S` on Windows/Linux) to open the menu.

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `swapenv.autoRefresh` | `true` | Auto-refresh on file saves |
| `swapenv.showNotifications` | `true` | Show success/error messages |
